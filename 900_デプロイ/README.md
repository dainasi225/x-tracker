# 900_デプロイ

このディレクトリは、配布・公開用の成果物を管理する場所です。

## 目的

- `001_開発` で検証済みの内容だけを、配布可能な形でまとめる
- 初心者向け配布で必要なファイル（実行手順、設定手順、リリースノート）を分離する

## 運用ルール

- 実装・修正は原則 `001_開発` で行う
- 配布時に必要なファイルのみをこのディレクトリへ反映する
- APIキーや秘密情報は置かない（`.env` やトークンの実値は禁止）

## 想定サブディレクトリ

- `releases/` : バージョン別の配布物
- `docs/` : 配布先向け手順書
- `checklists/` : リリース前チェック項目

上記3つは作成済みです。
